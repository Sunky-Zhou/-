## 方案标题
宠物喂食箱
## 方案应用场景
场景：宠物的运输过程
宠物在转移、运输或交易过程中健康问题难以保证。
通过本宠物箱，用户在旅行托运、交易运输或是独自外出时可以随时查看宠物状态，定时投喂食物，开关灯甚至语音交流。
大大方便对宠物的照料。
